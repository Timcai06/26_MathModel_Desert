# 26_MathModel_Desert

2026年校模拟赛2：B题“穿越沙漠”可复核求解工作区。

本目录把题面事实、六关地图、规则模拟、确定天气精确求解和 Result 回填拆开，避免“代码算出一个数，但无法证明逐日策略合法”。

## 当前完成情况

- 六关参数和地图已重建为规范数据，图结构测试覆盖连通性与邻接对称性。
- 单人逐日模拟器覆盖移动、停留、挖矿、沙暴、采购、负重、现金、失败和终点结算。
- 第一、二关已构造成时间扩展网络混合整数规划，由 HiGHS 证明全局最优。
- 第一关最优值为 `10470`，第 24 天到达；第二关最优值为 `12730`，第 30 天到达。
- 精确解已由独立模拟器逐日回放，并写入 `output/final/Result.xlsx`。
- 第三关精确在线 MDP 与第四关567候选选模已完成；新增保持天气边际不变的 Markov 持续性模型，并在两档边际、三档持续性构成的分布模糊集内完成鲁棒重选与集合外压力测试。
- 第四关新增固定240/240满负重、无补给子问题的精确 MDP—拉格朗日对偶基准：B5 是风险可行下界，对偶上界量化其策略缺口，且明确不外推为原问题全局最优。
- 第五关已由完整最佳响应动态规划求得纯策略均衡；第六关已完成三人交互模拟、角色条件二维装载校准，以及每名玩家 `70条最短路 × 2种风险可行错峰` 的受约束最佳响应检验。
- 46页完整论文采用严格黑白竞赛版式，正文系统展开MILP、Bellman递推、鲁棒阈值、风险对偶区间、Markov分布模糊集、多人联合转移、最佳响应和统计区间推导。
- 15张正式图均同时提供矢量PDF与300 dpi PNG；新增天气持续性压力、角色二维装载可行域和偏离可利用度图。
- 附录包含六类可复现算法、第一/二关完整逐日流水、风险预算与天气持续性复核、可证伪清单，以及从真实仓库读取的九组核心建模代码。
- 问题二、三的结论分别见 `docs/问题2阶段报告.md` 与 `docs/问题3阶段报告.md`，最终论文见 `output/final/穿越沙漠策略研究_完整稿.pdf`。

## 目录

- `B题/`：原始题面、附件和空白 Result 模板，只保存原始材料。
- `src/desert/scenarios.py`：六关参数、天气与邻接图。
- `src/desert/simulator.py`：独立规则模拟器。
- `src/desert/optimizer.py`：第一、二关时间扩展整数规划。
- `src/desert/reference_plans.py`：公开轨迹的回归版本，不参与最优性证明。
- `scripts/solve_known_weather.py`：运行精确求解并输出 JSON 流水。
- `scripts/fill_result.py`：把 JSON 流水写入原始 Result 模板的副本。
- `data/`：后续天气样本、规范参数和实验输入。
- `output/problem1/known_weather_exact.json`：HiGHS 最优解的逐日审计记录。
- `output/final/Result.xlsx`：当前可交付结果表。
- `output/final/穿越沙漠策略研究_完整稿.pdf`：完成编译与逐页验收的最终论文。
- `output/problem2/`：第三、四关的精确/仿真结果。
- `output/problem3/`：第五关均衡与第六关样本外比较。
- `output/robustness/`：天气持续性验证、第四关分布鲁棒重选、第六关角色装载与可利用度证据。
- `docs/深入调研提示词.md`：可直接交给 Deep Research 的完整提示词。
- `model_versions/`：模型版本、实验口径与晋级记录。
- `paper/`：论文正文、图和表的唯一归档位置。
- `tests/`：地图与模拟器测试。

Overleaf 协作时只需上传 `paper/` 目录中的内容，并以 `main.tex` 为主文档、XeLaTeX 为编译器；`paper/code/` 已包含附录引用的代码副本。

## 复现

```bash

make analyze-problem2
make optimize-level4
make analyze-level4-gap
make analyze-problem3

make result

```

运行 `make all` 可重跑六关主链、测试、图表、论文、Excel与最终交付复制。

命令应在“数模26spr-校模拟赛2”根目录运行。精确求解器依赖 `highspy==1.15.1`；项目内虚拟环境已安装该版本。

## 一项已发现的数据质量问题

公开 Result 表的第二关第 15—18 天水量分别写成 `110, 86, 56, 26`，但这些天在矿山挖矿，按题面高温/沙暴三倍消耗应为 `100, 76, 46, 16`。第19日移动消耗后再购买 `201` 箱水和 `86` 箱食物，日末水量为 `201` 箱；现金减少 `3730=201×10+86×20`，与逐日流水完全一致，最终最优值仍为 `12730`。

本工作区的精确求解器还找到了与公开路线不同但收益相同的第二关最优策略，说明该关存在多重最优解。
