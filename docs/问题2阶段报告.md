# 问题2阶段报告：仅知当天天气的在线决策

## 1. 结论边界

题目只说明玩家在行动前知道当天天气，没有给出未来天气的概率分布。因此不存在脱离概率假设的唯一“期望最优策略”。本阶段采用两层口径：

1. 第三关：给定晴朗/高温的独立同分布概率，求有限时域精确 MDP；只要某种天气在分布支撑集中出现，其对应的全部天气序列都必须可行。
2. 第四关：把“较少出现沙暴”参数化为沙暴概率和预留沙暴天数，比较稳健阈值策略的收益与失败风险。

天气在每天行动前揭示。策略只能使用日期、当前位置、现金、水、食物、当天气象和既往历史，不得读取未来天气。

## 2. 第三关：精确有限时域 MDP

### 2.1 状态和 Bellman 递推

天气揭示前的状态为

\[
s_t=(t,l_t,w_t,f_t),
\]

其中现金不影响可行性，因为第三关没有村庄，现金变化只来自挖矿，可作为即时收益累加。当天观察天气 \(z_t\) 后，在移动、停留、挖矿中选择合法行动。终点价值为现金加剩余物资半价回收价值，截止日未到终点的价值为负无穷。

程序对所有整数箱库存、位置、日期和当天气象进行精确 Bellman 搜索，并枚举所有可行初始采购。随后对全部 \(2^{10}=1024\) 条晴朗/高温序列逐条回放，核对 Bellman 期望值和规则合法性。

### 2.2 结果

| 高温概率 | 初始水/食物（箱） | 期望终值（元） | 5%分位数 | 最坏值 | 失败序列数 |
|---:|---:|---:|---:|---:|---:|
| 0.2 | 54 / 54 | 9382 | 9270 | 9190 | 0 / 1024 |
| 0.5 | 54 / 54 | 9310 | 9190 | 9190 | 0 / 1024 |
| 0.8 | 54 / 54 | 9238 | 9190 | 9190 | 0 / 1024 |

三档概率下的稳健期望最优策略相同：初始购买水54箱、食物54箱，沿最短路 `1→4→6→13` 连续行走三天，不去矿山。54箱恰好可以承受三天全部高温的行走消耗。

这个结论不是说挖矿永远无利可图，而是说：绕矿路线需要五个移动日，比直达多两个移动日；只要高温有正概率且要求所有支撑内天气序列不失败，就必须为高温绕行预留昂贵资源，抵消了晴天挖矿收益。

## 3. 第四关：鲁棒阈值策略与系统选模

### 3.1 比较策略

- `直达_B4`：按8个高温移动日加4个沙暴停留日准备物资，非沙暴日走最短路，沙暴日停留。
- `挖矿_B4`：初始满载水240箱、食物240箱，先走最短路到矿山；只有在保留“回终点所需高温移动资源+剩余4个沙暴日资源”后仍安全时才挖矿。
- `挖矿_B6`：与上一策略相同，但按6个沙暴日留出时间和资源，风险更低。

每种天气模型使用相同随机种子生成5000条配对样本。随后对沙暴预算3--9、
初始水量180--280和食物量180--280（步长10）做容量与资金过滤，共得到567个
合法候选。选模阶段每种分布使用1000条配对天气；先要求基准失败率不超过0.2%、
不利分布失败率不超过0.5%，再最大化不利分布下将失败记为0元后的总体终值。
最终测试改用全新随机种子，每个“策略×天气分布”使用20000条天气。

### 3.2 样本外结果

| 天气模型 | 策略 | 失败率 | 平均终值（元） | 5%分位数 | 平均到达日 |
|---|---|---:|---:|---:|---:|
| 低沙暴（5%） | 直达_B4 | 0 | 7846 | 7625 | 8.42 |
| 低沙暴（5%） | 挖矿_B4 | 0 | 10774 | 9035 | 12.33 |
| 低沙暴（5%） | 挖矿_B6 | 0 | 9926 | 8800 | 11.34 |
| 基准（10%） | 直达_B4 | 0 | 7794 | 7555 | 8.91 |
| 基准（10%） | 挖矿_B4 | 0.12% | 10584 | 8915 | 12.66 |
| 基准（10%） | 挖矿_B6 | 0 | 9746 | 8083 | 11.69 |
| 不利（15%） | 直达_B4 | 0.16% | 7698 | 7470 | 9.41 |
| 不利（15%） | 挖矿_B4 | 0.92% | 10064 | 8720 | 12.70 |
| 不利（15%） | 挖矿_B6 | 0.10% | 9263 | 7923 | 11.81 |

### 3.3 系统搜索与独立样本外结果

网格搜索入选参数为初始水240箱、食物240箱、沙暴预算5天。独立测试结果如下。

| 天气模型 | 失败率 | 95% Wilson区间 | 总体均值（元） | 95%均值区间 | 总体5%分位（元） |
|---|---:|---:|---:|---:|---:|
| 低沙暴 | 0 | [0, 0.0192%] | 10358.8 | [10345.4, 10372.3] | 8880.0 |
| 基准 | 0.055% | [0.0307%, 0.0985%] | 10162.9 | [10149.7, 10176.1] | 8805.0 |
| 不利 | 0.385% | [0.3082%, 0.4809%] | 9635.3 | [9621.1, 9649.5] | 8627.5 |

失败样本按0元计入总体均值与分位数，因此不含幸存者偏差。基准分布下，入选策略
比原 `挖矿_B6` 的总体均值高约418元；不利分布下高约392元，同时满足预设风险预算。

### 3.4 推荐口径

在附件“30天内较少出现沙暴”的语义下，推荐系统搜索所得的 `挖矿_B5` 作为
第四关主策略。原 `挖矿_B4` 可作为高风险对照，原 `挖矿_B6` 更保守但总体收益较低。

该推荐是基于明确天气假设的数值结论，不是无条件全局最优。若论文要求“绝不失败”，必须进一步给出沙暴总数上界；若沙暴可以任意多次，任何策略都无法保证在固定截止日前到达。

### 3.5 固定装载子问题的精确最优性区间

为单独衡量 B5 的动态决策质量，固定其实际装载为水240箱、食物240箱，并禁止村庄补给（该装载恰好满负重）。在完整5×5地图、30天期限和移动/停留/挖矿动作空间内，用天气揭示后的精确 MDP 求解，并对失败率不超过0.2%施加拉格朗日对偶上界。

| 量 | 精确结果 |
|---|---:|
| B5 总体期望值 | 10161.64 元 |
| B5 精确失败率 | 0.0307% |
| 风险约束最优值的对偶上界 | 11239.66 元 |
| 可证区间宽度 | 1078.02 元（上界的9.59%） |

这说明 B5 在该共同子问题中是满足风险预算的可行下界，距风险约束动态最优的可证缺口不超过9.59%。该结论不外推为“原第四关全局最优”，因为完整问题还允许改变初始装载与村庄补给。

## 4. 可复现文件

- `src/desert/online.py`：天气先揭示、策略后决策的执行接口。
- `src/desert/level3_mdp.py`：第三关精确 MDP。
- `src/desert/level4_policies.py`：第四关阈值策略。
- `src/desert/level4_mdp.py`：固定装载无补给子问题的精确MDP与拉格朗日对偶基准。
- `src/desert/weather.py`：天气分布、采样和全序列枚举。
- `src/desert/evaluation.py`：失败率、收益分位数和到达日评估。
- `output/problem2/level3_mdp_sensitivity.csv`：第三关精确结果。
- `output/problem2/level4_policy_comparison.csv`：第四关样本外比较。
- `output/problem2/summary.json`：完整机器可读结果。
- `src/desert/level4_search.py`：567个第四关候选的配对搜索。
- `scripts/optimize_level4.py`：训练/测试种子隔离和最终选模流水。
- `output/problem2/level4_optimization.json`：风险预算与独立测试结果。
- `output/problem2/level4_exact_mdp_gap.json`：B5可行下界与风险约束对偶上界。

运行：

```bash
make analyze-problem2
make optimize-level4
make analyze-level4-gap
```
