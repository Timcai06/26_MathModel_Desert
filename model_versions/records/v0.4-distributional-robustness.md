# v0.4：天气持续性、角色装载与六关偏离预言机

## 目标

修复 v0.2/v0.3 只在 IID 天气下选模的边界，并把第六关从有限策略对比推进到显式、可复核的单边偏离搜索。

## 模型与口径

- 天气转移：`P_rho = rho I + (1-rho) 1 p^T`，保持边际分布 `p` 不变。
- 分布模糊集：基准/不利两档边际，`rho = 0, 0.15, 0.35`；`rho = 0.70` 仅作集合外破坏性测试。
- 风险预算：基准 0.2%，不利 0.5%；最终验收使用 95% Wilson 上界。
- 失败样本终值：0 元。
- 随机种子基准：`20261021`，天气验证、训练、校准、预言机和测试使用不同派生随机流。

## 晋级结果

- 第四关：240/240 箱、B9；不利边际且 `rho=0.35` 时，30000 条样本失败率 0.1967%，Wilson 上界 0.2536%，总体均值 8033.4 元。
- 第六关：角色装载为 215/215、215/215、225/230，错峰为 0、0、1 天；同一压力点玩家失败率 0.3400%，固定角色最大 Wilson 上界 0.4344%，总体均值 7383.3 元。
- 六关偏离类：每名玩家完整枚举 70 条最短路和 2 种风险可行错峰装载，共 140 个候选；独立样本外最大类约束可利用度为 0 元。

## 边界

- `rho=0.70` 下第四、六关均可能越过风险预算，不属于当前保护集。
- 六关偏离类不含绕矿、村庄补给和任意历史依赖策略，因此只能称为类约束零可利用度，不能称完整 Markov 完美均衡。
- 第四关鲁棒训练每个模糊集点使用 500 条配对序列；最终结论以冻结参数后的每点 30000 条独立测试为验收依据。
